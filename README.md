<p class="has-line-data" data-line-start="18" data-line-end="21">一、一般信息<br>
在开始前，你需要了解的关于游戏的信息有：<br>
请先确保自己的配置足够运行新维加斯。并且与一般游戏不同的是，这里建议你不仅要满足官方给出的最低要求，最好是能接近推荐的配置要求。一方面作为一款9年前的游戏，如果只以较低画质运行，其画面表现力将十分不堪。更重要的是新维加斯在低画质选项下曾产生了很多BUG，一些MOD也会需要你将贴图质量调到高以上。因此建议你在购买下载前关注一下配置要求。</p>
<blockquote>
<p class="has-line-data" data-line-start="21" data-line-end="22">当然通过本指南，你可以最大化发挥硬件的资源。所以只要你的配置不是太差，莫哈维废土都可以任你驰骋。</p>
</blockquote>

这一部分主要针对Win10系统，Win7可以跳过：<br>
如果你正在使用远古版本的Win10，建议你更新到较新的系统。但不建议使用最新的Win10系统，可能会存在某些未知bug。</p>
<blockquote>

<p class="has-line-data" data-line-start="44" data-line-end="45">Win10系统可以设置优化的选项很多，网上有很多关于此的指南，不在这里重复，主要说三点比较重要的：</p>
<ul>
<li class="has-line-data" data-line-start="45" data-line-end="46">关闭系统自带的游戏模式，对于新维加斯来说该模式会造成不稳定</li>
<li class="has-line-data" data-line-start="46" data-line-end="47">调整你的输入法，Win10自带的输入法影响了很多游戏，如果你不能保证游戏过程中不误触，最好的方法是单独添加英语作为第二键盘，并按alt+shift切换。</li>
<li class="has-line-data" data-line-start="47" data-line-end="49">是关掉通知，以及其他可能在后台弹出对话框/提示的软件。新维加斯在切换窗口时卡死可能会让你疯掉，如果你必须要频繁切回桌面，后面针对此会有单独的步骤，但总的来说建议在游玩新维加斯的过程中尽量不要切出去。</li>
</ul>
<p class="has-line-data" data-line-start="49" data-line-end="51">对于笔记本电脑，请确保电源选项选择正确，如高性能或卓越性能。而对于所用用户，请更新到较新的显卡驱动，并保证有足够的硬盘空间。<br>
<strong>如果你之前安装过辐射新维加斯游戏后卸载，为保证本指南的能正常安装，请确保删除所有游戏文件、相关存档及注册表残余。</strong> 方法是使用搜索功能搜索Fallout NV相关的文件及文件夹然后删除，并运行注册表清理工具删除相关的残余条目。</p>
<blockquote>
<p class="has-line-data" data-line-start="51" data-line-end="52">注册表清理工具推荐 CCleaner 和 Wise Registry Cleaner</p>

<p class="has-line-data" data-line-start="53" data-line-end="56">三、游戏安装<br>
1.安装参数：<br>
所有modding工具必须安装在所有默认Windows文件夹之外（例如Program Files，Program Files（x86）和Documents）。Fallout New Vegas和Mod Organizer 2必须安装在同一个盘上，但不要将Mod Organizer 2和mod文件夹安装在Fallout New Vegas根目录下。</p>
<p class="has-line-data" data-line-start="57" data-line-end="61">2.常用术语：<br>
根文件夹(根目录)：<em>Steam / steamapps / common / Fallout New Vegas</em> 或 <em>GOG Galaxy / Games / Fallout New Vegas</em><br>
Data文件夹：<em>Steam / steamapps / common / Fallout New Vegas / Data</em> 或 <em>GOG Galaxy / Games / Fallout New Vegas / Data</em><br>
MO2：Mod Organizer 2（将很快安装）</p>
<p class="has-line-data" data-line-start="62" data-line-end="65"><strong>3.禁用叠加层：</strong><br>
需要为Fallout New Vegas禁用任何叠加以防止内存泄漏。一些具有覆盖的常见应用程序是Steam，GOG，NVIDIA GeForce Experience，Discord和MSI Afterburner。<br>
如Steam的禁用方法为：在库中右键点击Fallout：New Vegas，选择 属性，在 常规 选项卡下找到 在游戏中启动Steam界面 并反勾选它。这一步会使你无法使用Steam自带的截图功能，完成成就时也不会有右下角弹窗，但是还是非常建议你不要启动这一项，新维加斯游戏内有成就提示，后面也会介绍其他截图方式。</p>
<p class="has-line-data" data-line-start="66" data-line-end="70">4.设置Fallout New Vegas Modding文件夹：<br>
您的modding文件夹将充当您安装的所有工具的主目录，以及任何备份/屏幕截图等。从现在开始，此文件夹将被称为Modding文件夹。<br>
在与Fallout New Vegas安装相同的盘上的默认Windows文件夹之外的所选位置创建一个新文件夹，注意在根文件夹夹之外。将它命名为任何你喜欢的，如 <em>Fallout NV Modding</em> ，并创建两个子文件夹名为 <em>Backups</em> 和 <em>Tools</em><br>
备份整个游戏，将Fallout New Vegas文件夹复制到刚才创建的Backups中</p>
<p class="has-line-data" data-line-start="71" data-line-end="74">5.生成新鲜的.INI文件：<br>
从根文件夹运行FalloutNVLauncher.exe<br>
弹出 Detecting Video Hardware 的窗口，单击确定。之后弹出窗口提示程序根据你的硬件已设置画质，单击确定。</p>
<ul>
<li class="has-line-data" data-line-start="74" data-line-end="76">如果没有任何弹出窗口，请导航到 <em>文档/My Games/FalloutNV</em> 并删除以.INI结尾的所有文件，然后重新运行FalloutNVLauncher.exe</li>
</ul>
<p class="has-line-data" data-line-start="76" data-line-end="81">选择 Options<br>
确认游戏检测到并正确使用你的GPU，并选择了合适画质。如果没有正确识别GPU，请手动选择并调整画质选项，最好以Medium或High为初始选项。<br>
根据你的偏好设置分辨率（启动器通常将其设置为低于最大分辨率的一个）<br>
注意 Antiailiasing 项可以设置为8 Samples（启动器最多将其设置为4 Samples）<br>
单击 OK，然后单击 EXIT 退出</p>

<li class="has-line-data" data-line-start="240" data-line-end="241">它允许修改比通过本机游戏设置可访问的更多设置，并提供每个设置及其功能的说明。</li>
