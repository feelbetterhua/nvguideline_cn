<p>本指南旨在建立一种一站式的入门指导，帮助你了解游玩新维加斯需要的准备工作。并整合最新的优化配置选项，使你以最快的速度以及最丝滑的方式体验废土的旅程。<br>本指南不仅适合刚接触新维加斯的新手，也适合远离沙场已久，准备回归的老兵。因为一些Mod和优化方式的发展已与之前的方式不相兼容，因此本指南将会是你最好的选择。<br>本指南内容主要基于外网最近比较全面的一个MOD指南Viva New DDR3</li><li>硬盘：128GB SATA3.0 SSD</li><li>系统：Win10 LTSC 2019</li></ul> <p>目前的配置及优化可以使原版游戏在1080P，最高画质下以60帧稳定运行。即使加了大量画质MOD后依旧可以在高画质，900P的条件下运行5个小时不跳出。我觉得这样的游戏体验才是2019年的新维加斯应该有的。<br>并且必须一提的是，网上流传了很久新维加斯不适合在Win10上运行。以目前的情况看已经没有那么不堪，但前提是你的Win10版本不能太旧，也不能太新。我的LTSC是基于1809的，可以完美运行。 </p><p>一、一般信息<br>在下载前，请你先确保自己的配置足够运行新维加斯。并且与一般游戏不同的是，这里建议你不仅要满足官方给出的最低要求，最好是能接近推荐的配置要求。一方面作为一款9年前的游戏，如果只以较低画质运行，其画面表现力将十分不堪。更重要的是新维加斯在低画质选项下曾产生了很多BUG，一些MOD也会需要你将贴图质量调到高以上。因此建议你在购买下载前关注一下配置要求。<br>当然通过本指南，你可以最大化发挥硬件的资源，改善原版游戏优化糟糕的现象，提升运行的稳定性与流畅度，所以只要你的配置不是太差，莫哈维废土都可以任你驰骋。<br>要声明的是，本指南初期只针对原版游戏，不会添加或修改游戏内容。如果你是你想通过MOD改变游戏玩法或内容，抑或是需求画质的提升，请等待后期我的更新，或直接阅读原MOD指南。 </p><p>二、准备工作<br>在运行游戏之前，你需要什么？当然是游戏本身。不论网上流传着什么版本，我在这里只推荐Steam或GOG的终极版，也就是全DLC的版本。纯净的正版是本指南适用的唯一版本，如果是保证完全纯净的学习版应该也可以，但我不能保证其他版本也有效果。<br>除此之外你还需要一个MOD管理器，也许对于本入门指南来说，MOD管理器并没有发挥太大左右，但熟悉MOD管理器将为你未来的游戏旅程很有帮助。本指南使用Mod Organizer 2，如果你之前没有使用过或使用较老的MOD管理器，建议你更新到Mod Organizer 2，这是目前最先进的MOD管理器，拥有很多优良的特性。<br>因为本指南专门针对想玩中文的玩家，你肯定还需要一个中文补丁。目前只推荐民间汉化第5版，这是在天邈汉化出来前(虽然都不知道还到底会不会出来)最完善的，也是安装最简单的一个版本。注意它只支持终极版或者升级了UP7并且安装了所有dlc的版本。<br>其他需要的准备工作还有：最新的显卡驱动，足够的硬盘空间以及了解INI文件(实际上就是文本文档)的修改。 </p><p>三、系统设置<br>这一部分主要针对Win10系统，Win7可以跳过。 </p><ul> <li>首先关闭系统自带的游戏模式，对于新维加斯来说该模式会造成不稳定</li></ul> <ul> <li>其次调整你的输入法，Win10自带的输入法影响了很多游戏，如果你不能保证游戏过程中不会切换到中文，最好的方法是单独添加英语作为第二键盘，并按alt+shift切换。</li></ul> Browse ,在弹出的框中导航到 Mod Organizer 2 文件夹并双击选择 ModOrganizer.exe<br>5.在 INI Path 一栏下拉菜单中选择您现在使用的Mod Organizer 2配置文件，如 ModOrganizer&gt;Defaul 。在弹出的对话框中选择 NO ，此时BethINI会重新启动，等待其重启即可。<br>6.重新启动BethINI后导航到 Basic 选项卡，在 Presets 栏中选择 BethINI Presets ，再选择 Recommended Tweaks ，程序会自动优化相应的INI文件，之后请根据配置在 Poor 到 Ultra 中选择合适的画质选项。<br>7.选择完画质后在 Display 一栏中选择合适的分辨率，如果没有需要的分辨率则可以手动输入。可以自行选择 Antialiasing 和 Anisotropic Filtering 倍数(一般使用预设即可)。在右边选择 Allow 3.0 Shaders ，其他保持不变。<br>8.最后选择下方的 Enable File Selection ，确定一切选择正确后点击 Save and Exit 退出。等待程序关闭后导航到 <em>文档/My Games/FalloutNV</em> 打开 RendererInfo.txt 。查看下方 Shader Package 后的数字，如果是19，则不需要进一步修改。如果不是19，请导航到 <em>data/Shaders</em> 文件夹找到序号对应Shader Package后数字的文件，如 shaderpackage013.sdp ，将其命名为 shaderpackage013.sdp.backup 。<br>之后把 shaderpackage019.sdp 复制一份到其他位置，将复制的shaderpackage019.sdp重命名为shaderpackage013.sdp，再剪切回原文件夹。整个过程即是备份正在使用的shaderpackage，并将shaderpackage019.sdp重命名以强制启用。 </p><blockquote> <p>强制启用shaderpackage019以打开Shaders 3.0，这应该会增加游戏帧数及稳定性。</p> </blockquote> </body> </html>
